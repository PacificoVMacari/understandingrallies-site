Understanding Rallies privacy approach.

This section can be expanded into a full privacy policy page before Microsoft Store submission. The wording below is a suitable starting point for a local-first desktop app.

Privacy summary

UR Squash is designed to process squash footage and rally data locally on the user's device. Video files and analysis outputs are not uploaded to Understanding Rallies servers by default.

  • Users choose which video files or session files to analyse.
  • Analysis outputs are stored locally on the user's computer.
  • No account is required for the basic desktop workflow.
  • Support enquiries can be sent using the contact email below.
